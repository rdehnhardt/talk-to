# /talk-to:setup

Initialize the project folder structure for Talk To.

Create the following folders with an empty `INDEX.md` in each. Skip any that already exist.

Folders to create:
- `business-rules/INDEX.md`
- `tasks/INDEX.md`
- `decisions/INDEX.md`
- `schemas/INDEX.md`
- `design/INDEX.md`
- `infra/INDEX.md`
- `marketing/INDEX.md`
- `financial/INDEX.md`

Each `INDEX.md` should contain:

```markdown
# <Folder Name>

| File | Summary | Status | Date |
|------|---------|--------|------|
```

Also create `ROADMAP.md` and `STATUS.md` at the project root if they don't exist.

`ROADMAP.md`:
```markdown
# Roadmap

> Managed by PM. High-level phases and milestones.

## Phases

_No phases defined yet. Use `/talk-to:pm` to start planning._
```

`STATUS.md`:
```markdown
# Project Status

> Last updated: <today's date>

## Current Focus
_Not defined yet._

## In Progress
_None._

## Blocked
_None._
```

After creating, list what was created and confirm the project is ready.
